## 📄 Pages Description

### 🏠 Home Page (index.html)
- Header with contact info
- Navigation menu
- Sidebar with news
- Banner image
- Course listing section
- "How We Work" section
- Footer with links & social icons

### ℹ️ About Us
- Institute information
- Training & project details
- Benefits list

### 📚 Courses
- 3-column grid layout
- 9 course cards

### 🖼️ Gallery
- 3x3 image grid
- Hover zoom effect

### 📝 Enquiry
- Full enquiry form
- Inputs, dropdowns, radio buttons

### 📞 Contact Us
- Phone, email, address
- Map section
